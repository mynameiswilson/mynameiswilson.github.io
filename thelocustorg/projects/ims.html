2002.04.21

the very, very, very first and uber-beta release of InstantManServant. get it here. I’ve included toc.py v1.1 in there for the moment -- the first stable release will not include this, and you’ll have to go get the latest. also there is little or no documentation at the moment -- give me time! It’s pretty simple, tho, and if you wanna write a module, just call it mod$whatever.py, put it in your ims directory, add a [mod$whatever] section to the config file, and put active=1 under it. voila. use the "moduptime.py" file as an example. make sure you change the trigger value, too.
